Common Meals App

NSS Front-End Capstone

Front-end capstone project: designed for the residents of Germantown Commons to facilitate the scheduling of common meals. Users (residents) can schedule a meal, be assigned to a cook team when scheduling a meal, and sign-up to attend a meal.

read about the tech++

The app was built using Angular and implements full CRUD. The pages are designed using Boostrap grids and styled using SASS. Technologies include HTML, CSS, SASS, ES6 JavaScript, Bootstrap, AngularJS, and Grunt.

Address Book

Simple address book application

read about the tech++

This app was built using HTML, ES6 JavaScript, AngularJS, Bootstrap, CSS, SASS, Grunt; implements CRUD. Uses AngularJS to load "address book" records from Firebase JSON file and output the data to the DOM. Initially built using jQuery; then refactored using AngularJS, and added User Auth, Registration, Login, Logout.
